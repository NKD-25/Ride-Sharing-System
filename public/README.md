# RideShare — React frontend

This folder contains a Vite + React conversion of the original static frontend.

Quick start

1. Open a terminal in `client/`:

```powershell
cd client
npm install
npm run dev
```

2. Open the dev server (usually http://localhost:5173).

Build for production

```powershell
npm run build
npm run preview
```

Notes

- The app uses localStorage for demo data (no backend). Data is stored in the browser.
- If you previously ran npm in the `public/` folder, run the commands above from `client/` instead.
- The original static site remains in `public/` for reference.
